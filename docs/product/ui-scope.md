# MacDockShell UI Scope

Цель первого этапа: собрать визуальную оболочку интерфейса в стиле Apple Dock/Menu Bar для Windows 11 на Qt Quick/QML.

Что входит в MVP интерфейса:
- прозрачное полноэкранное окно-оверлей без рамки
- верхняя компактная menu bar-панель
- нижний центрированный dock с увеличением иконок при hover
- плавающая glassmorphism-подложка
- область для системных статус-иконок справа
- заглушка Apple-style меню слева
- базовые анимации появления, hover и bounce
- тёмная палитра с мягкими световыми акцентами

Что пока не входит:
- интеграция с Win32 API
- список реальных открытых приложений
- drag-and-drop файлов и ярлыков
- автозапуск, tray, виртуальные рабочие столы
- live-превью окон, pin/unpin, context menu

Архитектурная цель этапа:
- сразу заложить QML-структуру так, чтобы позже подключить C++ backend через модели и bridge-объекты.
